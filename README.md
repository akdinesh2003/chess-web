# Chess Web App

A beautiful, modern chess web application with multiple themes, sound effects, move history, hints, and a powerful AI bot (Stockfish) for hard mode.

---

## Features
- Play chess in your browser with a stunning UI
- Multiple board themes (Classic, Neon, Royal, Forest)
- Human vs Human, or play against the computer (Easy, Medium, Hard)
- Hard mode uses the Stockfish chess engine for professional-level play
- Move history, hints, sound effects, and more

---

## How to Run

### 1. **Clone or Download the Repository**
```
git clone https://github.com/akdinesh2003/chess-web.git
```
Or download and extract the ZIP.

### 2. **Download Stockfish.js**
- Go to [https://github.com/niklasf/stockfish.js/releases](https://github.com/niklasf/stockfish.js/releases)
- Download the `stockfish.js` file (about 1MB)
- Place it in the same folder as `index.html`

### 3. **Run a Local Server**
> **Important:** Stockfish will only work if you run your site from a local server (not by double-clicking index.html)

#### **Using Python:**
1. Open a terminal in your project folder
2. Run:
   ```
   python -m http.server
   ```
3. Open your browser and go to [http://localhost:8000](http://localhost:8000)

#### **Using VS Code Live Server:**
1. Open the folder in VS Code
2. Right-click `index.html` â†’ "Open with Live Server"

### 4. **Play Chess!**
- Open your browser to the local server URL
- Play against a friend or the computer
- For the strongest bot, select "Hard" (requires Stockfish.js)
